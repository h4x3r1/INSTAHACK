### installation

````bash

apt-get update

apt-get upgrade

apt-get install git nano

apt-get install python3

apt-get install python3-pip

git clone https://github.com/h4x3r1/instahack

cd instahack

pip install -r requirements.txt

python3 ihack.py

````

### commands

command for Kali Linux, Debian and Termux Android.

````bash

python3 ihack.py

````
